# 同步/异步

`Gex`支持两种执行方式

- 同步
- 异步

::: warning 注意
`同步`执行是默认选项
:::

## 同步

同步执行的意思是，在调用命令时一定要等到命令执行完成才会返回
::: warning 注意 同步执行会阻塞当前`goroutine`
:::

<<< @/../example/guide/sync.go

## 异步

异步执行的意思是，在调用命令时，不会等到命令执行完成就返回
::: warning 注意 异步执行通常会很快返回，如果程序需要在外部程序特定条件下才能继续往下执行的话，请配置`检查器`一起使用来完成该目的
:::
