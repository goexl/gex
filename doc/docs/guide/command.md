# 命令

可以这样理解一个命令
> 一个短时间内运行的功能

比如说

- 帮助
- 数据迁移
- 打印版本信息
- 其它应用程序所需要的命令

建议

- 命令别名是一个数组，可以是任何你想要的别名，区分大小写
- 命令上下文使用`pangu.app.Context`，可以使用该上下文来获取所有的命令信息
- 命令的执行可以是长期的（一直不退出），完全按用户方的使用来定
